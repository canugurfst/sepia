Sepia Image Converter
The Sepia Image Converter is a Python script that applies a sepia filter to JPEG images. This script takes an input JPEG image and generates a sepia-toned version of the image.

Features
Converts JPEG images to sepia-toned versions.
Simple and easy-to-use Python script.
Usage
Make sure you have Python installed on your system.

Place your input JPEG image in the same directory as the sepia.py script.

Open a terminal or command prompt.

Navigate to the directory containing the sepia.py script and your input image.

Run the script using the following command:

python sepia.py

The sepia-toned image will be generated in the same directory with the filename output.jpg.
